### РЕКОМЕНДАЦИИ 💡

нужно использовать списки это тег ul

ссылки не должны вести на картинки достаточно поставить #

лучше использовать классы при верстке а не id  особенно если элементы пере используемые

использование в ссылках атрибута target="_blank" открывает новое окно но нужно что бы ссылка ввела на текущую страницу  (#)

применения before  покрывает не всю карточку можно использовать вот так  где inset: 0; растягивает на всю ширину

```javascript
main a::before {
  content: "";
  position: absolute;
  inset: 0;
  background: linear-gradient(to top, rgba(0,0,0,0.5), transparent);
  border-radius: 10px;
}
```
