### РЕКОМЕНДАЦИИ 💡

- В css в многих местах указан - font-style: Semi Bold; Font-style может принимать только такие значения - [https://developer.mozilla.org/ru/docs/Web/CSS/Reference/Properties/font-style](https://developer.mozilla.org/ru/docs/Web/CSS/Reference/Properties/font-style)
- letter-spacing не может быть в % - [https://developer.mozilla.org/ru/docs/Web/CSS/Reference/Properties/letter-spacing](https://developer.mozilla.org/ru/docs/Web/CSS/Reference/Properties/letter-spacing)
- leading-trim переименовано в text-box-trim [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/text-box-trim](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/text-box-trim) И поддержка свойства очень ограничена - [https://caniuse.com/?search=text-box-trim](https://caniuse.com/?search=text-box-trim) просто игнорируется браузером. Лучше не использовать.
- У #banner в width и height нет единиц измерения (скрин).
Свойства просто игнорируются.
- Любой section по правилам семантики должен иметь заголовок. В `<section class="icon">` его нет. Лучше заменить тег `<section>` на div или nav с aria-label
- Используете одинаковые кавычки по всему проекту. Сейчас есть места, где часть с двойными. например, class="cards-img", а часть в формате class='status'
- Поправить табуляцию и в css и в html (пример на скрине)
- Лучше использовать стили к классам вместо тегов (типа, main #promotion p). При использовании классов, компонент изолируется, стили легко переопределить, нет проблем со специфичностью. Если вы измените p на что-то другое, стили не будут работать.
- Рекомендуется добавлять width и height для всех картинок в html разметке. Так браузер будет резервировать место для картинки - [https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/img#:~:text=Use%20both%20width%20and%20height](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/img#:~:text=Use%20both%20width%20and%20height),to%20mitigate%20content%20layout%20shifts.
- В tel: нужно убрать пробелы и скобки
- Для всех img нужен alt. Не важно, что это иконка корзины 🙂
